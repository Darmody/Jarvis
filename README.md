# Jarvis

> 这个项目将搭建一个属于我家的超级管家，文档及工程将随着进度推进不断完善。

**项目结构发生重大改变，web server端将使用Koa.js框架，按app形式管理业务**

## Components

### Jarvis

> 大管家，负责一切家务调度与人员派遣

- io.js
- koa.js

### Tyrion

>  管家副手，负责管理家里的所有物资

- NFS
- aria2

### Poppy

>  管家副手，负责接待主人与来宾

- Polymer
- Material Design
